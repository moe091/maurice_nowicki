PORTFOLIO


White Glove - Website

White Glove is a website I made for a cleaning service. I designed it in photoshop and then converted it to html/css/javascript. I also wrote a simple Java program for the client to use to edit the sites content

Gulp - HTML5 Mobile-Browser Game

Libra2 is math learning game I was hired to create that incorporates realtime graphics and physics on the android platform.

Libra2 - Android Game

Libra2 is math learning game I was hired to create that incorporates realtime graphics and physics on the android platform.

Let's Play - Android App

Let's Play is an Android App I ported from the iOS platform for a client. It's the app version of the book the published "Let's Play - 100 Card, Dart, Domino and Dice Games." It features all the content from the book plus a multi-functional scoreboard for keeping score in various different types of games.
